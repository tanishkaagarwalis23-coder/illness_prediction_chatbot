# Illness Prediction Chatbot

## Overview
A machine learning-powered chatbot that predicts likely diseases based on user symptoms.

## Features
- Symptom-based disease prediction
- Top 3 disease recommendations
- Follow-up symptom questions
- Streamlit web interface
- Random Forest classifier

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- Joblib

## Live Demo
<https://illness-prediction-chatbot-8imu.onrender.com>

## Dataset
Disease-Symptom dataset with 4,900+ records and 130+ symptom features.
